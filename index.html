🚨 Breaches & Active Threats
Notable

"SleeperGem" supply-chain attack hijacks dormant RubyGems maintainer accounts

📅 Jul 19, 2026

Aikido Security researcher Charlie Eriksen found a new gem, git_credential_manager, built over nine hours across four rapid releases into a working dropper that fetches a payload from a lookalike Forgejo host and quietly skips CI environments to target developer laptops instead. The same publisher pushed malicious updates to two other long-dormant gems, including fastlane-plugin-run_tests_firebase_testlab with 574,661 downloads under a different maintainer entirely.

Why it matters: RubyGems had mostly avoided the account-takeover wave that's hit npm and PyPI for over a year — a reactivated account with zero recent activity is exactly the kind of "boring" account defenders should flag, not ignore, across every package registry your build pulls from.

Volexity: SonicWall SMA1000 zero-days were exploited for root access weeks before disclosure

📅 Jul 17, 2026

Volexity is tracking a previously undocumented threat actor, UTA0533, that chained CVE-2026-15409 (CVSS 10.0 SSRF) and CVE-2026-15410 to plant a setuid root-run binary on SMA1000 appliances as early as June 22 — nearly a month before SonicWall's public advisory. The group then injected a custom webshell (ORANGETAIL) and the open-source Suo5 proxy tool into a legitimate SonicWall process to tunnel traffic and harvest credentials.

Why it matters: "Patch by the federal deadline" doesn't retroactively evict an attacker who's had root since before the CVE existed — if you run SMA1000, treat this as a confirmed-compromise investigation, not a patch-and-move-on item, and hunt for Volexity's published IOCs.

🛡️ Vulnerabilities & Patches
Critical

15-year-old nginx bug enables pre-auth remote code execution (CVE-2026-42533)

📅 Jul 19, 2026

The flaw has been silently exploitable since March 2011, when nginx's map directive gained regex support: a shared, mutable capture-state array can be overwritten mid-request, causing a heap buffer overflow and an ASLR-defeating pointer leak. F5 shipped fixes in nginx 1.30.4/1.31.3 and NGINX Plus R36 P7/37.0.3.1; researchers achieved 10-for-10 exploitation on Ubuntu 24.04 with ASLR enabled and are withholding full PoC details for 21 days post-patch.

Why it matters: A CVE this old, this widely deployed, and this reliably exploitable is a "patch this week, not this quarter" situation — nginx sits in front of an enormous share of the internet's web traffic, and the 21-day disclosure grace period is a rapidly closing window, not a reason to wait.

"MemGhost" research: a single email can plant persistent false memories in AI agents

📅 Jul 13, 2026

Researchers behind the "When Claws Remember but Do Not Tell" paper built a reinforcement-learning framework that converts one poisoned email into a planted, hidden "fact" an agent later treats as true — succeeding in 87.5% of background-mode runs against OpenClaw on GPT-5.4 and 71.4% against a Claude Code SDK agent on Sonnet 4.6, across 56 fresh test cases.

Why it matters: Memory-enabled agents that persist "facts" across sessions create a new class of long-lived poisoning that a one-time prompt-injection fix won't catch — any agent with write access to its own memory store needs that store audited like a config file, not trusted like a cache.

Notable

"PromptFiction" flaw let a single click auto-submit hidden prompts in Claude Desktop

📅 Jul 15, 2026

Oasis Security found that Claude Desktop's claude:// URL handler skipped the review step the web app enforces, so a crafted link could open the app and submit an attacker's full prompt automatically — concealed below a "show more" fold — with no confirmation screen. Combined with filesystem-connected MCP servers, the flaw could plant persistence in files a user later asked Claude to edit; Anthropic fixed it in Claude Desktop 1.1.2321.

Why it matters: This is Oasis's second Claude Desktop disclosure this year (after "Claudy Day" in March) following the same pattern — a convenience feature that skips a confirmation step becomes an attacker's automatic-execution primitive; audit any custom URI-scheme handler your AI desktop tools register.
