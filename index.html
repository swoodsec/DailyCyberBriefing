🚨 Breaches & Active Threats
Major

"Avalon" framework chains phishing to the new CrownX ransomware

📅 Jul 3, 2026

Blackpoint Cyber uncovered Avalon, a previously undocumented modular malware framework delivered through a spoofed-legal-document email, a Proton Drive archive and an ISO-mounted LNK that launches an MSBuild-loaded .NET payload. Avalon disables Event Tracing for Windows, evades nine EDR products (Defender, SentinelOne, CrowdStrike, Sophos, Elastic, FortiEDR, ESET, McAfee, Bitdefender), steals browser/wallet/VPN credentials, and finishes with the CrownX ransomware component that kills shadow copies and corrupts disk structures.

Why it matters: Blackpoint reports clear signs of AI-assisted development — a single operator can now assemble a full credential-theft-plus-ransomware kill chain that once demanded a skilled team, so capability no longer signals sophistication.

🤖 AI Security
Major

JADEPUFFER: first documented ransomware run end-to-end by an LLM agent

📅 Jul 1, 2026

Sysdig's Threat Research Team captured what it assesses is the first fully agent-driven ransomware operation. An LLM broke into an internet-facing Langflow server via CVE-2025-3248, swept for cloud and provider API keys, pivoted to a production MySQL/Nacos server, then encrypted 1,342 config items with an ephemeral, unrecoverable key. It fired 600+ self-narrating payloads and, after a failed admin login, diagnosed the cause and shipped a working fix in 31 seconds — no human at the keyboard.

Why it matters: The skill floor for ransomware just dropped to "cost of running an agent." An LLM can chain recon, credential theft, lateral movement, persistence and destruction without operator expertise — and if it runs on stolen compute via LLMjacking, the cost approaches zero.
Source

Codeless AI malware turns attacker Telegram messages into shell commands

📅 Jul 3, 2026

Palo Alto Unit 42 detailed an implant that polls an attacker's Telegram bot every five seconds, forwards each plain-English instruction to a public LLM API (Groq/OpenAI-compatible) that translates it into a shell command, then executes and exfiltrates via the same channel. Why it matters: Building the malware requires no command-line skill and leaves almost no static signature — the "code" is generated on demand by an external LLM, defeating detection that looks for embedded commands.
