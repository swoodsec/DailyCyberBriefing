"CrashStealer" macOS malware poses as Apple's own crash-reporting tool

📅 Jul 14, 2026

Jamf Threat Labs tracked CrashStealer from a May sample through early-July in-the-wild detections: a C++ infostealer delivered via a notarized, Gatekeeper-passing dropper that harvests browser profiles, roughly 80 cryptocurrency wallet extensions and 14 password managers, then encrypts the stolen files with AES-GCM before exfiltration to slow analysis.

Why it matters: A notarized dropper that clears Apple's own trust checks undercuts the assumption that "Apple-signed" means "safe," and the client-side encryption of stolen data is a maturing evasion trend worth tracking across other stealer families.

Hackers backdoor Jscrambler's npm package with a Rust infostealer targeting AI coding tools

📅 Jul 13, 2026

A threat actor used compromised publishing credentials to push malicious jscrambler npm releases (8.14–8.20) downloaded nearly 1,500 times before Jscrambler deprecated them two hours later. The payload, obfuscated with ChaCha20-Poly1305 encryption per Socket's analysis, hunted for cloud credentials, CI/CD tokens, crypto wallets, and configs for AI coding tools including Claude, Cursor and Windsurf.

Why it matters: The two-hour window shows how fast a compromised publishing credential can weaponize a trusted package — and the deliberate targeting of AI coding-tool configs marks a new, high-value theft target developers may not think to rotate.

Critical

ClaudeBleed-linked flaw in Claude for Chrome still unpatched across eight releases, Manifold says

📅 Jul 14, 2026

Manifold reported two flaws to Anthropic on May 21, related to the mitigation shipped after the original ClaudeBleed disclosure: a malicious extension can fake the "real user click" Anthropic's fix requires to trigger Claude actions, and in the extension's autonomous "Act without asking" mode this proceeds with no visible warning, exposing Gmail, Docs and Calendar. None of the eight Claude for Chrome versions released since — including the latest 1.0.80 — appear to fix it.

Why it matters: A mitigation that survives eight patch cycles without closing the actual gap is a case study in how hard it is to retrofit trust boundaries onto an agentic browser extension after the fact — treat "fixed" claims for agent products skeptically until independently verified.

Notable

"Ghostcommit" hides prompt injection inside PNG images to steal repo secrets from AI coding agents

📅 Jul 11, 2026

Researchers at UMKC's ASSET group built a pull request that points an AGENTS.md convention file at an innocuous-looking PNG; the exploit text lives inside the image, which text-based reviewers like CodeRabbit and Bugbot never open. A coding agent later reads the picture, encodes a repo's .env secrets as integers, and commits them as a harmless-looking constant. Cursor and Antigravity leaked secrets across multiple models; Claude Code refused every time.

Why it matters: The coding harness, not the underlying model, determined whether the attack succeeded — a strong argument for evaluating the security posture of the AI coding tool itself, not just which model it's running.
