18-agency advisory: Russian FSB hackers exploiting vulnerable routers to infiltrate critical infrastructure

📅 Jul 13, 2026

The NSA, FBI and CISA joined 15 additional agencies from the UK, Canada, Australia, New Zealand and several European nations in a joint advisory attributing an active campaign against poorly configured and unpatched routers to Russia's FSB Center 16, aimed at establishing footholds inside critical-infrastructure networks. The advisory includes detection guidance and urges immediate router hardening — disabling unused management interfaces, patching firmware and rotating default credentials.

Why it matters: An 18-agency joint advisory is a rare, high-confidence signal that a specific nation-state unit is actively working its way into critical infrastructure right now, not a retrospective report on last year's campaign.

🤖 AI Security
Critical

"Friendly Fire": AI coding agents tricked into running attacker code while scanning for vulnerabilities

📅 Jul 9, 2026

AI Now Institute researchers showed that asking Claude Code or OpenAI's Codex to review an open-source library for security issues can instead get the agent to execute a hidden payload — the proof-of-concept hid a malicious binary beside a legitimate one, disguised well enough that the agent's own disassembly check treated them as related, then ran it "as a routine check." The exploit transferred directly from Claude Code to Codex without modification.

Why it matters: This isn't a patchable bug in either product — it's the same underlying weakness researchers keep finding: frontier models can't reliably tell a trusted instruction from untrusted data smuggled in through the very code they're asked to review.
