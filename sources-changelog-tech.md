# Tech Sources — Changelog

This file logs how the **Tech** source roster (`sources-tech.json`) evolves over time. The daily 6 AM briefing task vets general-technology outlets (big tech & business, devices & consumer products, software & platforms, science & emerging tech), adds up to 3 newly qualifying sources per day (deduped by domain), and records every change here. Seeded core sources are never removed. This roster is a sibling to the Cyber and AI briefings and deliberately avoids narrow cybersecurity and pure AI-model coverage.
