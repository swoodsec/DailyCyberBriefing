🚨 Breaches & Active Threats
Major

Attackers actively probing critical Gitea Docker auth-bypass, 13 days after disclosure

📅 Jul 6, 2026

Sysdig detected the first in-the-wild exploitation attempts against CVE-2026-20896 (CVSS 9.8), a flaw where Gitea's official Docker images shipped with "REVERSE_PROXY_TRUSTED_PROXIES = *" hard-coded, letting any unauthenticated internet client spoof the X-WEBAUTH-USER header and gain elevated access — no password, token or session cookie required. Roughly 6,200 internet-facing Gitea instances remain exposed; Gitea fixed the default in versions 1.26.3/1.26.4.

Why it matters: A misconfigured default in an official container image turned a routine self-hosted Git server into a one-header account takeover — patch or isolate exposed Gitea Docker deployments now, before probing turns into exploitation.

Major

Maximum-severity SimpleHelp bypass (CVSS 10.0) actively deploying Djinn Stealer

📅 Jun 30, 2026

Attackers are exploiting CVE-2026-48558, a critical SimpleHelp RMM auth bypass where a forged OIDC token grants a fully authenticated "Technician" session — even bypassing MFA, since technicians can self-register a new MFA method on first login. Confirmed intrusions deploy TaskWeaver, an obfuscated Node.js loader, followed by the new Djinn Stealer, which targets cloud tokens, Git/SSH credentials, Docker configs and AI dev-tool keys (Claude, Gemini, Codex) across Windows, macOS and Linux. CISA added it to KEV with a July 2 FCEB patch deadline; roughly 1,000 of 14,000 internet-facing servers remain unpatched.

Why it matters: Remote-support tools are a favorite ransomware foothold precisely because they're trusted and privileged — and the stealer's specific hunt for AI-coding-tool API keys shows attackers now treat developer AI credentials as first-class loot.

SOCRadar's Threat Research Unit linked the FortiBleed campaign — which harvested 110M+ credentials from 430,000 FortiGate firewalls via a Go tool abusing FortiOS's own diagnostic command — directly to the INC Ransom and Lynx operations. An operator was found logged into both groups' negotiation panels, and at least 12 ransomware deployments trace to FortiBleed-derived access; a leaked internal document exposed a structured ~20-person crew.

Why it matters: This closes the loop from mass credential theft to actual encryption — every unrotated FortiGate credential from this campaign is a live ransomware foothold, not a theoretical risk.

🛡️ Vulnerabilities & Patches
Critical

Unpatched Argo CD repo-server flaw can hand attackers a full Kubernetes cluster

📅 Jul 6, 2026

Synacktiv published full technical details of an unauthenticated RCE in Argo CD's repo-server component after reporting it to maintainers in January 2025 and waiting 18 months with no fix and no CVE assigned. The repo-server's internal gRPC service has no authentication; a crafted GenerateManifest request can substitute a malicious script for the expected helm/kustomize call. Helm-chart installs leave network policies off by default, so an attacker who compromises just one pod can reach the repo-server, read the cluster's Redis password from an environment variable, and poison deployment data.

Why it matters: GitOps control planes now deserve tier-zero protection — with no patch available, the only defense is turning on Kubernetes network policies to isolate the repo-server and Redis ports immediately.

High

Citrix patches six NetScaler flaws; one already under attack within 24 hours

📅 Jul 6, 2026

Citrix fixed six NetScaler ADC/Gateway CVEs (CTX696604) rated 6.9–8.8, including CVE-2026-8451, a CitrixBleed-style pre-auth out-of-bounds memory read triggered by malformed SAML authentication requests, and CVE-2026-8452, a Gateway memory overflow enabling denial-of-service. Fixes landed in versions 14.1-72.61 and 13.1-63.18 and FIPS equivalents; CVE-2026-13474, an HTTP/2-request DoS, needs an additional manual configuration change to fully close. Active exploitation of CVE-2026-8451 began less than a day after disclosure.

Why it matters: NetScaler's SAML-parsing bugs keep echoing the CitrixBleed pattern that fueled mass ransomware intrusions in 2023 — the sub-24-hour exploitation window leaves almost no grace period to patch.

Why it matters: A patch that shipped quietly in May is only now getting the KEV spotlight it needed — a reminder to cross-check vendor advisory updates, not just initial release notes, when prioritizing patching.

🤖 AI Security
Major

Hidden web prompts trick 4 of 26 tested AI agents into sending crypto payments

📅 Jul 6, 2026

Zscaler's ThreatLabz documented two live indirect prompt-injection campaigns: one buries CSS-hidden and JSON-LD-embedded instructions in a fake "developer API key" page that tells visiting AI agents to pay 0.0012 ETH to an attacker wallet; the other impersonates the DeFi tracker DeBank via a typosquat domain reinforced with hidden prompts telling agents it's legitimate. Both use SEO poisoning to rank the pages highly so agents find them. Running its own agent across 26 LLMs, ThreatLabz got 4 to execute the fraudulent payment and 2 to misidentify the typosquat as genuine — including variants of Llama and Gemini.

Why it matters: This is prompt injection turned into a working monetization loop with no human victim required — any agent given browsing and payment capability is now a target for content it was never meant to trust.

Notable

JADEPUFFER: the first ransomware run end-to-end by an AI agent — that lost its own key

📅 Jul 1, 2026

Sysdig's Threat Research Team captured what it assesses as the first fully agentic ransomware operation. An LLM agent broke into an internet-facing Langflow instance (CVE-2025-3248), then autonomously enumerated the host, swept for cloud and LLM-provider secrets, moved laterally and ran a destructive database-extortion playbook — adapting in real time with self-narrating payloads. It encrypted 1,342 Nacos configs, but the key was printed once to a log and never saved, making the data permanently unrecoverable.

Why it matters: This is machine-speed extortion with no human in the loop — and the lost-key finale shows agentic attackers can be simultaneously more capable and more brittle, destroying data they meant to ransom.
